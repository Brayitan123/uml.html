UML (Lenguaje Unificado de Modelado)

Booch et al. (2021):

"El Lenguaje Unificado de Modelado (UML) es un lenguaje visual estándar para la especificación, construcción y documentación de los artefactos de sistemas de software. Proporciona una forma de visualizar la arquitectura de un sistema utilizando un conjunto estandarizado de símbolos y diagramas" (p. 24).

Rumbaugh et al. (2017):

"UML es un lenguaje gráfico para visualizar, especificar, construir y documentar sistemas de software intensivo. Ofrece un estándar para describir un 'plano' del sistema, incluyendo aspectos conceptuales como procesos de negocio, funciones del sistema, y aspectos concretos como expresiones de lenguajes de programación, esquemas de bases de datos y componentes reutilizables" (p. 12).
